# Embedded-projects

In this repository I will be showcasing my projects in Embedded Systems. They range from simple LED Lights to a Space invaders game. 
All projects have demos and source code. 

### PROJECT 1: 
First part is to make sure LED lights work and can be controlled using 2 buttons (Left and Right). Second part is using the buttons to count up or down from 1 to 10 in binary.

### PROJECT 2:
Using potentiometers for 3 purposes: first, output potentiometer output into the output terminal. Second, add a 8 segment display and output the potentiometer output within the range of 1-5, or A-E. Then, adding checks and using buzzer to make a noise when the potentiometer displayed value is E or 5. 

### Project 3
Using potentiometer and LED light, and using the clock cycle to change the color of the LED lights. set checks for the LED to change from red and blinking, to yellow, to green.

### Project 4
Using the sonar and LED light to measure distance from sonar to a object. As the object moves, the distance displayed in the 4 digit segment display increases, until it reaches FULL. LED light changes color depending on distance.

### Project 5
Using 8 segment display, joystick and motor. As user moves joystick left or right, 'r' or 'l' is displayed, and motor spins in the correct direction. 

### Project 6
6 LED lights display the direction the motor is spinning. 2 buttons control the speed of the motor. Pressing the left button makes it spin slower, and pressing the right button makes it spin faster. This is done by controlling the clock cycle of the motor. Servo is also used, when joystick is moved up or down. Different uses of the joystick causes different outputs.

### Custom Project 
I developed a simple Space Invaders game. demos and code are included
